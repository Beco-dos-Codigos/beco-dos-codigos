Começando do absoluto zero

Instalar Python, entender terminal, pip, venv e escrever os primeiros programas.

Esta trilha ensina Python começando literalmente do começo: instalar Python, abrir terminal, entender o que é pip, criar ambientes virtuais e escrever os primeiros códigos.

01
instalação

Instalando Python corretamente

Como instalar, validar e entender o ambiente.

1.1 Verificando se Python está instalado ▾

Windows

Terminal
python --version

Alternativa moderna

Terminal
py --version

Por que existe o comando py?

O launcher py ajuda quando existem múltiplas versões de Python instaladas no Windows.

Verificando pip

Terminal
python -m pip --version

O que é pip?

O pip é o gerenciador de pacotes do Python. Ele instala bibliotecas criadas por outras pessoas.

02
terminal

Entendendo comandos básicos

O que significa -m, help e como Python executa módulos.

2.1 O que significa -m? ▾

O parâmetro -m diz ao Python: “execute um módulo Python”.

Exemplo
python -m pip install requests

Boa prática

Prefira python -m pip em vez de apenas pip. Isso evita problemas com múltiplas instalações de Python.

Ajuda integrada

Terminal
python --help
python -m pip --help
03
pip

Instalando bibliotecas

Instalar, atualizar e usar requirements.txt.

3.1 Instalando pacotes ▾
Instalar requests
python -m pip install requests
Atualizar pip
python -m pip install --upgrade pip
Instalar requirements.txt
python -m pip install -r requirements.txt

O que é requirements.txt?

Um arquivo que lista dependências do projeto. Outros desenvolvedores podem te enviar esse arquivo. Você instala tudo de uma vez com -r.

04
venv

Ambientes virtuais

Isolando dependências corretamente.

4.1 Criando um .venv ▾
Criar ambiente virtual
python -m venv .venv
Ativar no Windows
.venv\Scripts\activate
Desativar
deactivate

Pense assim

Um ambiente virtual é como uma caixa isolada. Cada projeto guarda suas próprias bibliotecas.
